# Vault Explorer

Vault Explorer is a highly advanced, natively integrated desktop application designed explicitly for managing, browsing, and optimizing your local video library with lightning-fast speeds. Functioning as an "Extrovert Explorer," it is completely localized and non-destructive.

## 🚀 Key Features

* **Asynchronous Lightning Indexing:** Built safely on native Node `fs.promises` generators to recursively crawl massive folder structures in seconds without blocking the UI.
* **Virtual "Fake" Folders Lifecycle:** Organize your videos virtually! Create completely abstracted, local metadata-driven "fake folders" allowing you to drop files entirely virtually across sessions without actually touching or modifying your actual raw OS files!
* **Hover-Play WebM Chunking & Video Overlays:**
  * Say goodbye to static thumbnails. Generating previews binds directly to ffmpeg, intelligently carving out mathematical chunks of dynamic `<video>` preview tracks!
  * On mouse hover, an intelligent, hardware-accelerated preview modal automatically overlays your thumbnails seamlessly inside the HTML framework.
* **Smart Global Renaming:** Press `F2` to trigger an intelligent local filename regex that seamlessly patches the video file alongside all attached posters, thumbnails, and generated sub-assets symmetrically!
* **Built-In HTML5 Media Execution:** Instantly double-click complex arrays to summon a borderless, darkened viewing theater powered intrinsically by Chromium HTML5 logic for fluid, hardware-accelerated viewing.
* **Memory Scroll Persistence:** Remembers precisely what Virtual directory and Y-offset you were located inside perfectly across browser reboots.
* **"Everything" Search API Sync:** Instantly hooks `es.exe -size` backend indexes for lighting-fast directory meta-blob size evaluations.
* **Native Multi-Select Mechanics:** Includes pixel-perfect, relative-DOM intersecting drag-boxes matching Explorer's lasso framework flawlessly.

## 📦 Tech Stack

* **Electron:** Chromium framework powering the engine seamlessly natively.
* **Javascript & Node.js:** High octane asynchronous backends bridging internal DOM nodes correctly to robust standard protocol systems (IPC).
* **WinUI 3 Glass Design Aesthetics:** Styled fundamentally identically with dynamic acrylic aesthetics, dark modes, blur backdrops, and active smooth animations.

## 🔨 Development

1. Start Dev Environment: `npm start` (or `electron .`)
2. Package Windows Executable (NSIS): `npm run dist`

Enjoy a hyper-organized Vault!
