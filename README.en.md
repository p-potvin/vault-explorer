# 📂 Vault Explorer

Vault Explorer is a highly advanced, natively integrated desktop application designed explicitly for managing, browsing, and optimizing your local video library with lightning-fast speeds. Functioning as an "Extrovert Explorer," it is completely localized, secure, and non-destructive.

---

## 🚀 Key Features

* **Asynchronous Lightning Indexing:** Built safely on native Node `fs.promises` generators to recursively crawl massive folder structures in seconds without blocking the UI.
* **Virtual "Fake" Folders Lifecycle:** Organize your videos virtually! Create completely abstracted, local metadata-driven "fake folders" allowing you to drop files entirely virtually across sessions without actually touching or modifying your raw OS files.
* **Hover-Play WebM Chunking & Video Overlays:** On mouse hover, an intelligent, hardware-accelerated preview modal automatically overlays your thumbnails seamlessly inside the HTML framework. Previews are generated by binding to FFmpeg.
* **Smart Global Renaming:** Press `F2` to trigger an intelligent local filename regex that seamlessly patches the video file alongside all attached posters, thumbnails, and generated sub-assets symmetrically.
* **Built-In HTML5 Media Execution:** Instantly double-click complex arrays to summon a borderless, darkened viewing theater powered intrinsically by Chromium HTML5 logic for fluid, hardware-accelerated viewing.
* **Picture-in-Picture (PiP) Mini-Player:** Seamlessly toggle a persistent, non-blocking Picture-in-Picture mini-player overlay to keep watching your media while browsing other folders.
* **Automated Next Video Countdown:** Facilitates hands-free playlist viewing with a configurable automatic next-video transition countdown timer on video completion.
* **⚡ Real-Debrid URL Downloader:** High-speed unrestricting & downloading for Rapidgator, Turbobit, and magnet links, with built-in Secure Proxy Tunneling (e.g., GreenCloud VPS via Tailscale).
* **ASR Speech & Translation Benchmark:** Native GPU CUDA and Simulator benchmarking suites for evaluating speech processing latency and system performance.
* **Real-Time Livestream Translation Pipeline:** A state-of-the-art speech-to-speech spoken translation pipeline designed to translate English streams to French in real-time.
  * **Dual Ingestion Modes:** Supports standard system WASAPI audio loopback (capturing browsers, VLC, or music players) or direct HTTPS media streams (via robust FFmpeg piping).
  * **Advanced AI Tech Stack:** Uses local, GPU-accelerated NVIDIA NeMo `ParakeetV3` for transcription and local `Kokoro-82M ONNX` for premium 32-bit float speech synthesis.
  * **Acoustic Volume Ducking:** Integrates `pycaw` to automatically duck active browser/app volumes to 15% during synthesized speech playback, eliminating audio feedback loops.
  * **Sci-Fi Waveform Visualizer:** Animates a real-time, glowing 20-band audio peaks visualizer directly mapped to Kokoro's synthesis buffer.
  * **Custom Voice & Accent Profiles:** Adjust voice presets, customize silence detection sensitivity, calibrate mixing volume levels, and map standard European French or localized Québécois accent profiles.
* **Bilingual Interface Persistence:** Supports persistent, bilingual English/Québécois toggle settings and persistent custom theme managers (e.g., Golden Slate, WinUI 3 Glass).

## 🛠️ Standalone PowerShell Utilities

Vault Explorer ships with a collection of high-performance, standalone PowerShell wrappers situated in the `powershell/` directory. These allow you to execute heavy background jobs, benchmarks, audio normalization, and live translation directly from the terminal. 

Every script automatically resolves its required virtual environment (`vault-explorer/.venv` for lightweight loopback services and `vaultwares-media-processing/.venv` for heavy neural net tasks) to guarantee seamless executions.

### 1. Start-LivestreamTranslator.ps1
Launches the system WASAPI loopback translation pipeline, capturing real-time playback from your browser or audio player, translating from English to French, and playing synthesized Kokoro TTS.
* **Location:** `powershell/Start-LivestreamTranslator.ps1`
* **Parameters:**
  * `-Voice` (String): Kokoro TTS voice preset. Defaults to `ff_siwis`. Other options include `af_bella`, `af_nicole`, `ff_siwis`, `bf_emma`.
  * `-Lang` (String): Target translation language format (e.g., `fr-fr` or `fr-ca`). Defaults to `fr-fr`.
  * `-Threshold` (Double): Amplitude noise gate threshold to trigger transcription. Defaults to `0.005`.
  * `-Volume` (Double): Synthesis audio playback mixing volume multiplier (from `0.0` to `1.0`). Defaults to `0.80`.
* **Example:**
  ```powershell
  .\powershell\Start-LivestreamTranslator.ps1 -Voice "af_bella" -Threshold 0.008 -Volume 0.90
  ```

### 2. Start-StreamTranslator.ps1
Pipes a live external HTTPS audio/video stream through FFmpeg directly into the translation and speech synthesis pipeline.
* **Location:** `powershell/Start-StreamTranslator.ps1`
* **Parameters:**
  * `-StreamUrl` (String, **Mandatory**): Direct HTTPS/RTMP media stream source URL.
  * `-Voice` (String): Kokoro TTS voice preset. Defaults to `ff_siwis`.
  * `-Lang` (String): Target language format. Defaults to `fr-fr`.
  * `-Threshold` (Double): Amplitude noise gate threshold. Defaults to `0.005`.
  * `-Volume` (Double): Output volume multiplier. Defaults to `0.80`.
* **Example:**
  ```powershell
  .\powershell\Start-StreamTranslator.ps1 -StreamUrl "https://example.com/live/stream.m3u8" -Voice "bf_emma" -Volume 0.75
  ```

### 3. Start-AudioNormalization.ps1
Processes a targeted video file, isolates background tracks using Demucs, runs ASR voice transcription, generates SRT subtitles, and encodes the modernized stereo audio mix back into a preservation copy.
* **Location:** `powershell/Start-AudioNormalization.ps1`
* **Parameters:**
  * `-VideoPath` (String, **Mandatory**): Absolute path to the source video file.
  * `-VaultRoot` (String, Optional): Parent folder or vault directory. Defaults to the video file's parent folder.
  * `-Transcribe` (Switch): If specified, automatically runs transcription and outputs bilingual `.srt` sidecars.
  * `-TranslateTo` (String, Optional): Target language code to synthesize SAPI translation tracks.
* **Example:**
  ```powershell
  .\powershell\Start-AudioNormalization.ps1 -VideoPath "C:\Videos\movie.mp4" -Transcribe -TranslateTo "fr"
  ```

### 4. Start-AsrBenchmark.ps1
Runs full performance profiling on the local transcription systems, warm-loading model weights and outputting high-fidelity latency telemetry.
* **Location:** `powershell/Start-AsrBenchmark.ps1`
* **Parameters:**
  * `-Native` (Switch): Force benchmarks to load the native NVIDIA NeMo weights over CUDA.
  * `-ForceSimulation` (Switch): Enforces offline simulator mode to profile without downloading model weights.
* **Example:**
  ```powershell
  .\powershell\Start-AsrBenchmark.ps1 -Native
  ```

### 5. Start-PreviewGenerator.ps1
Recursively walks down a directory structure and compiles hover-preview continuous WebM clips and keyframe thumbnails for all discovered videos.
* **Location:** `powershell/Start-PreviewGenerator.ps1`
* **Parameters:**
  * `-ScanDir` (String, Optional): Absolute path to target scan folder. Defaults to drive `F:\` or system root folder.
* **Example:**
  ```powershell
  .\powershell\Start-PreviewGenerator.ps1 -ScanDir "C:\MyVault"
  ```
